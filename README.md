# ドゥラメンテ产驹数据库

这是一个用于浏览大鸣大放（Duramente / ドゥラメンテ）产驹资料的静态网页数据库。

公开访问地址：

https://dacron92mr.github.io/duramente-progeny-database/

## 网站内容

目前收录 2018-2022 年出生的 725 匹大鸣大放产驹。页面可以查看：

- 马名、英文名与香港译名
- 生年、性别、毛色、所属、练马师、生产牧场
- 母名、母父、母父大系统、牝系编号
- 奖金、主要胜鞍、战绩等级
- 单匹马的完整比赛记录
- 部分退役马的种马或繁殖牝马记录
- 种牡马成绩、母父分析、世代比较、距离与场地拆分
- 修正后的 Cross 祖先 / 祖先+Cross形式 / 结构层分析
- 牧场分析与赛马场胜率、连对率、前三率分析

## 使用方式

左侧筛选栏可以按生年、性别、毛色、所属、练马师、最高战绩、繁殖记录、母父个体、母父系和牝系筛选。

搜索栏支持输入日文马名、英文名、香港译名、母名、母父、练马师或牧场名。点击表格中的任意一匹马，可以打开详细资料页。

顶部导航可以切换到“種牡馬成績”“母父分析”“血統分析”“牧場分析”“競馬場分析”和“データ・方法”。分析页中的统计会同时显示分子、分母和样本量，避免只看百分比。

Cross 统计使用独立产驹数口径；同一匹马可以同时进入多个 Cross 组，所以各组奖金不能相加当作总奖金。赛次奖金字段目前覆盖率不足，网站暂时不展示赛次层面的奖金汇总。

## 数据说明

本数据库整理自公开赛马资料，主要用于个人兴趣研究和浏览展示。数据来源包括 netkeiba、JBIS、香港赛马会公开资料与 Japan Stud Book 公开页面等。

本仓库发布的是静态快照版本，所有资料已经导出为 JSON 文件，网页本身不连接后台服务器，也不会实时抓取数据。若原始数据库后续更新，需要重新导出并发布新版。

## 技术说明

这个站点是纯静态网页：

- `index.html`：页面结构
- `style.css`：页面样式
- `app.js`：搜索、筛选、排序、详情页交互
- `data/`：导出的马匹列表、详情、战绩、繁殖记录和分析指标 JSON

因为是 GitHub Pages 静态站点，朋友打开链接即可使用，不需要安装 Python、SQLite 或任何本地环境。
